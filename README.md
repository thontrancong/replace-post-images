# replace-post-images
Replace post images is a plugin for wordpress CMS.
Inside a post, download images from other domain, save its this domain and replace old image with new image.

# Install plugin

![alt text](https://goo.gl/photos/RmGnjcFdyu63F9Fz7)

After installed plugin, a button “Replace images” will appear in the form add/edit posts

![alt text](https://goo.gl/photos/CfkniamhjnCzQz7VA)

All images will save to uploads directory as other media
